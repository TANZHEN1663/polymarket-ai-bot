## 🚀 如何使用

### 步骤 1：获取 API 密钥

**Polymarket API**:
1. 访问 https://clob.polymarket.com
2. 注册账户
3. 获取 API Key、Secret、Passphrase

**AI API** (推荐 Qwen):
1. 访问阿里云百炼平台
2. 注册并创建 API Key
3. 选择 Qwen-plus 模型

### 步骤 2：配置

编辑 `polymarket_config.json`:
```json
{
  "polymarket": {
    "api_key": "你的 Polymarket API Key",
    "api_secret": "你的 Polymarket API Secret",
    "passphrase": "你的 Polymarket Passphrase"
  },
  "ai": {
    "provider": "qwen",
    "model": "qwen-plus",
    "api_key": "你的 Qwen API Key"
  },
  "trading": {
    "enabled": false,  // 先用模拟模式
    "mode": "paper"
  }
}
```

### 步骤 3：运行

**Windows**:
```powershell
cd c:\Users\a5414\nofx\polymarket
go run main.go
```

**使用启动脚本**:
```powershell
.\start.bat
```

**查看状态**:
```powershell
go run main.go -status
go run main.go -metrics
go run main.go -positions
```

## 📊 策略示例

### 价值投注策略工作原理

**场景**: Polymarket 市场"2026 年比特币能否达到 10 万美元？"

```
市场数据:
- YES 价格：58 cents (隐含概率 58%)
- NO 价格：42 cents (隐含概率 42%)
- 24h 交易量：$50,000
- 流动性：$25,000

AI 分析:
- 基于历史数据、市场趋势、外部因素
- AI 预测概率：70%
- 置信度：75%

分析结果:
- 差异：70% - 58% = 12% (显著差异)
- 期望值：0.12
- 策略：VALUE_BET
- 方向：BUY YES
- 仓位：$300 (Kelly 公式计算)
```
